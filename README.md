# 📊 Smart Analytics Tool

A Python-based data analytics dashboard developed in Google Colab for exploring and analyzing CSV datasets. The tool provides data previews, missing value analysis, statistical summaries, and multiple visualizations to help users gain insights from their data.

---

## 🚀 Features

- 📂 CSV Upload Functionality
- 👀 Dataset Preview
- ❓ Missing Value Analysis
- 📈 Statistical Summary
- 📊 Dynamic Data Visualizations
- 🔍 Trend and Pattern Identification

---

## 🛠️ Technologies Used

- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## 📁 Dataset

The project uses a CSV dataset containing:

| Column | Description |
|----------|------------|
| Country | Country Name |
| Year | Year of Observation |
| Births | Number of Births Recorded |

Example:

| Country | Year | Births |
|----------|------|----------|
| India | 2000 | 26000000 |
| USA | 2000 | 4000000 |
| China | 2000 | 17000000 |

---

## 📊 Visualizations

The dashboard generates the following visualizations:

### 1. Histogram
Displays the distribution of birth records.

### 2. Bar Chart
Shows total births by year.

### 3. Line Chart
Illustrates birth trends over time.

### 4. Scatter Plot
Shows the relationship between Year and Births.

### 5. Pie Chart
Represents each country's contribution to total births.

---

## 📈 Analysis Performed

### Dataset Preview
- View dataset structure
- Display sample records

### Missing Value Analysis
- Count missing values per column
- Visualize missing values using a heatmap

### Statistical Summary
Generate:
- Count
- Mean
- Standard Deviation
- Minimum
- Maximum
- Quartiles

---

## ▶️ How to Run

1. Open the notebook in Google Colab.
2. Run all cells.
3. Upload a CSV file when prompted.
4. Explore the generated statistics and visualizations.
